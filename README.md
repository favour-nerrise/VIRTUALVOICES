## How to run app?
Tested react app working with

```sh
$ git clone git@github.com:favour-nerrise/VIRTUALVOICES.git
$ cd main
$ npm install
$ npm start
```
### Requirements

* Atom, Sublime, VS Code, or any modern text editor
* Node.js and npm
